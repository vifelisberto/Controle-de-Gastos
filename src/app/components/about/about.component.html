Sobre Controle de Gastos

O nosso projeto tem objetivo dos usuários controlarem suas despesas no dia-a-dia, estudos, saúde, alimentação, viagens, etc.

Para facilitar essa tarefa, criamos um aplicativo de controle de despesas. A plataforma é útil para os interessados em gerenciar de forma mais eficiente os gastos e, assim, economizar.
